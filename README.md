## ASCII-Image-Converter

### Build
```Console
mkdir build
cd build/
cmake ..
make
```

### Usage
```Console
./aic <input_image_path> <output_image_path>
```

### Example
```Console
./aic ../res/test.jpg ../res/test_out.jpg
```
